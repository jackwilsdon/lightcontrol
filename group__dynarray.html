unsigned int dynarray_add ( dynarray_t *  array,
void *  item 
)

Adds an item to a dynamically sized array.

This method appends the provided item to the end of the array. If the new size of the array is greater than the current capacity then the array's capacity is doubled.

Parameters
[in,out]arrayThe array to add the item to.
[in]itemThe item to add to the array.
Returns
DYNARRAY_SUCCESS if the item was added successfully, DYNARRAY_ERROR otherwise.
unsigned int dynarray_free ( dynarray_t *  array)

Frees the memory used by a dynamically sized array.

This method performs the following actions on the provided array:

  • Frees the memory allocated to dynarray::data.
  • Sets the pointer of dynarray::data to NULL.
  • Sets the capacity of the array to zero.
  • Sets the size of the array to zero.
Parameters
[in,out]arrayThe array to free.
Returns
DYNARRAY_SUCCESS if the array was freed successfully, DYNARRAY_ERROR otherwise.

unsigned int dynarray_init ( dynarray_t *  array,
int  initial_capacity 
)

Initializes a dynamically sized array.

This method performs the following actions on the provided array

  • Allocates an array of pointers for dynarray::data.
  • Sets the initial capacity and current capacity to the capacity provided.
  • Sets the size to zero.

This method should not be called on an existing array, instead the array should be freed first.

Parameters
[in,out]arrayThe array to initialize.
[in]initial_capacityThe initial capacity of the array.
unsigned int dynarray_remove ( dynarray_t *  array,
int  index 
)

Removes an item from a dynamically sized array.

If the index provided is greater than the capacity of the array, DYNARRAY_ERROR is returned.

If the index provided is before the last item of the array it is removed and all items after it are shifted by one place to the left.

Parameters
[in,out]arrayThe array to remove the item from.
[in]indexThe index of the item to remove.
Returns
DYNARRAY_SUCCESS if the item was removed successfully, DYNARRAY_ERROR otherwise.

int *  size

Retrieves the size of a dynamically sized array.

The size of an array is the current number of item in the array.

This method uses dynarray::size internally but should be used instead of dynarray::size as it performs a NULL check.

Parameters
[in]arrayThe array to retrieve the size of.
[out]sizeThe size of the array.

DYNARRAY_SUCCESS if the size was retrieved successfully, DYNARRAY_ERROR otherwise.
